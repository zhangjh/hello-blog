# hello-blog
博客的源码仓库，写文章，更新等操作在此进行

hexo g生成public目录后，更新到线上

使用gulp对静态代码进行压缩，一键式部署脚本自动完成hexo编译、gulp压缩以及git上传操作。
由于github国内访问速度的问题，国内ip访问引流至gitcafe，国外ip访问引流至github。详情见参见`start.sh`外包脚本。

博客地址：[点击访问我的博客](http://zhangjh.me)
